# Jordan-Hospitals-Dataset
Locations of hospitals and health centers in Jordan - GIS dataset

#Description
This repository contains a geospatial layer of hospital locations. The dataset is intended for mapping, spatial analysis, and integration within GIS projects.

#Purpose
- Provide a spatial dataset of hospitals  
- Support GIS analysis and visualization  
- Enable use in planning, accessibility, and proximity studies  

#Data Format
- File format: Shapefile / GeoJSON / Feature Layer  
- Geometry type: Points and Polygons

# Attributes (if applicable)
Typical attributes may include:
- Hospital name  ( Arabic, English)
- Type/classification  
- Governorate
  
  Usage
1. Download or clone the repository  
2. Load the layer into GIS software (ArcGIS Pro)
